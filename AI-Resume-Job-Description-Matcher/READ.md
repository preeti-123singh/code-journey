## AI Resume and Job Description Matcher

## project overview
- Resume Parsing
- Job Description Parsing
- Skill matching
- Match percentage calculation

 ## Tech Stack
  - Python
  - HTML
  - NLP
 
 ## How to Run
 1. open terminal
 2. go to backend folder
 3. install requirements
    pip install -r requirements.txt
 4. Run:
    python app.py
 5. open browser at:
    http://.0.0.1:5000/
